# q-learning
Implementation of Q-learning on small grid-world

## Requirement

1. Have Python - we use python3.13 but I believe any, just not too old, would works with numpy.

2. Clone the repo.
```sh
git clone git@github.com:khanhdo05/q-learning.git
```

3. Create a virtual environment and install deps
```sh
python -m venv .venv
source ./.venv/bin/activate
pip install -r requirements.txt
```

4. Run `main.py`
```sh
python main.py
```
